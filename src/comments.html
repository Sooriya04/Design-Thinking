How to add a new field to an existing model in Django?

Asked by: Sarah, age 25, Software Engineer at Google

I have an existing model in Django, but I want to add a new field to it. What’s the best way to do this? I don’t want to lose any of the data that is already in the database.

Tags: Python, Django, Database

You can add a new field to an existing model in Django using the migrations framework. First, add the new field to your model definition. Then, create a new migration file and run the migration. This will update the database schema without losing any data.

Answered by: John, age 40, Senior Software Engineer at Facebook

In addition to what John said, make sure to specify a default value for the new field if it is required. Otherwise, you may encounter issues when running the migration. You can also use the `makemigrations` and `migrate` commands to generate and apply the migration.
